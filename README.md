# bidone
Test task

    Frontend: Angular 12
    Backend: .NET 5

Simple form to save first name and last name to json file

Launching:

    Prerequisites:
        - Node 14.15
        - Npm 6.14
        - Angular Cli 12.1

    Backend: 
        - Open solution under Backend
        - Restore nuget packages if required
        - Run
    Frontend:
        - Go to Frontend folder
        - Run "npm i"
        - Run "ng serve"
